# E-Commerce Sales Analysis

## Project Overview

This project analyzes the Brazilian Olist E-Commerce Dataset to identify sales trends, top-performing categories, regional performance, customer review patterns, and business opportunities.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Analysis Performed

### Data Cleaning
- Missing value analysis
- Duplicate checking
- Data type verification
- Datetime conversion

### Sales Analysis
1. Highest Revenue Product Category
2. Peak Sales Month
3. Best Performing State
4. Average Order Value Trend
5. Customer Review Score Distribution

### Visualizations
- Revenue by Category
- Monthly Sales Trend
- State Revenue Analysis
- Review Score Distribution
- Average Order Value Trend
- Product Category Revenue Analysis

### Dashboard KPIs
- Total Revenue
- Total Orders
- Top Category
- Best Month

### Business Insights
- 5 actionable business recommendations

## Dataset

Brazilian Olist E-Commerce Dataset (Kaggle)
